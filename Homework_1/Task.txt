## Чуть подробнее про метод update в репозитории

* Был:

```
User {
    uuid: d1744543-17c0-47ba-934c-ad8c2f1179d9
    usename: marsel
    password: qwerty007
}
```

* Вы делаете:

```
userSQL.setUsername(marat);
userSQL.setPassword(qwerty111);

userRepository.update(userSQL);
```

* Получаем:

```
User {
    uuid: d1744543-17c0-47ba-934c-ad8c2f1179d9
    usename: marat
    password: qwerty111
}
```

* В Java нельзя обновить кусок файла, нужно перезаписать его полностью. (но есть предположение от Камиля, что можно через randomAccessFile)
* Замечание от Аделя (можно обернуть в try-catch ошибку, связанную с ненайденным пользователем, чтобы избежать остановки приложения)
  * Такую обработку делаем в слое представления, потому что в разных видах представления ее нужно обработать по-разному
